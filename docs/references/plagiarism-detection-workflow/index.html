Plagiarism Detection and Prevention Guide

Zero Plagiarism Policy​

This module maintains a zero tolerance policy for plagiarism. All content must be original with proper attribution to sources.

What Constitutes Plagiarism​

Plagiarism includes but is not limited to:

  • Copying text directly from sources without attribution
  • Paraphrasing content without citation
  • Using ideas from sources without acknowledgment
  • Submitting someone else's work as your own

Originality Requirements​

  • All explanations must be written in your own words
  • Examples should be created specifically for this module
  • Code snippets should be original or properly attributed
  • Diagrams and figures should be created or properly licensed

Proper Attribution Methods​

Direct Quotes​

When quoting directly from a source:

  • Use quotation marks
  • Include in-text citation
  • Provide full reference in the bibliography

Example: According to the ROS 2 documentation, "DDS provides the underlying pub/sub communication layer" (ROS 2 Documentation, 2023).

Paraphrasing​

When expressing someone else's idea in your own words:

  • Clearly cite the original source
  • Ensure the meaning remains accurate
  • Do not change just a few words while keeping the structure

Ideas and Concepts​

When using ideas from others:

  • Acknowledge the source of the concept
  • Explain how it applies to your content
  • Cite appropriately

Tools for Plagiarism Detection​

  1. Turnitin - Comprehensive plagiarism checker
  2. Grammarly - Includes plagiarism detection
  3. Copyscape - Online content comparison
  4. SmallSEOTools - Free plagiarism checker

Workflow for Originality Verification​

  1. During Writing:

    • Keep track of all sources consulted
    • Note which ideas come from which sources
    • Write original explanations for technical concepts

  2. After Writing:

    • Run content through plagiarism detection tool
    • Verify all borrowed content is properly cited
    • Ensure all text meets originality standards

  3. Before Submission:

    • Final plagiarism check
    • Verify all citations are complete and accurate
    • Confirm all sources are in the reference list

Creating Original Content​

For Technical Concepts​

  • Explain concepts in your own words
  • Use original examples and analogies
  • Create unique diagrams or figures
  • Develop your own code snippets when possible

For Complex Topics​

  • Break down concepts into understandable parts
  • Use analogies relevant to the target audience
  • Create step-by-step explanations
  • Include practical applications

Citation Requirements​

All sources must be cited using APA format:

  • In-text citations for all borrowed content
  • Complete reference list at the end of each chapter
  • Consistent formatting throughout the module

Review Process​

Before finalizing content:

  1. Verify originality using detection tools
  2. Confirm all citations are present and correct
  3. Ensure all sources are properly referenced
  4. Check that content meets educational objectives
