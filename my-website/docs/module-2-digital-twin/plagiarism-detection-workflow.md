# Plagiarism Detection Workflow

This document outlines the workflow for ensuring all content in Module 2: The Digital Twin (Gazebo & Unity) meets the zero plagiarism tolerance requirement.

## Plagiarism Policy

All content must be original and rewritten. No direct copying of documentation text or other sources is permitted. All technical concepts must be explained in original language while maintaining accuracy.

## Detection Workflow

### 1. Pre-Writing Phase
- Identify all sources to be consulted
- Create source tracking document
- Plan how to synthesize information into original content

### 2. Writing Phase
- Write content in original language
- Paraphrase technical concepts rather than copying
- Use personal examples and explanations
- Attribute ideas to sources without copying text

### 3. Verification Phase
- Use plagiarism detection tools (e.g., Turnitin, Copyscape, Grammarly)
- Manually compare content against sources
- Verify all borrowed ideas are properly attributed
- Ensure all technical descriptions are in original language

### 4. Review Phase
- Have content reviewed by editors for potential plagiarism
- Cross-reference with original sources
- Verify all citations are properly formatted
- Confirm all content meets originality standards

## Tools for Plagiarism Detection

### Automated Tools:
- Turnitin: Comprehensive plagiarism detection
- Grammarly: Built-in plagiarism checker
- Copyscape: Web-based plagiarism detection
- SmallSEOTools: Free plagiarism checker

### Manual Verification:
- Side-by-side comparison with source materials
- Verification of paraphrasing accuracy
- Confirmation of proper attribution

## Standards for Originality

- 0% similarity score on plagiarism detection tools
- All technical concepts expressed in original language
- Proper attribution for all non-original ideas
- No direct quotes without proper citation and quotation marks

## Remediation Process

If plagiarism is detected:
1. Identify the plagiarized content
2. Rewrite in original language while preserving accuracy
3. Verify the rewritten content passes plagiarism checks
4. Update citations if necessary
5. Re-run plagiarism detection tools

## Documentation Requirements

- Maintain source tracking document
- Record plagiarism detection tool results
- Document any remediation actions taken
- Verify final content meets originality standards
